### Work with GitHub

- #### 產生 SSH key：

```
ssh-keygen -t rsa -C "your_email@youremail.com"
```

- #### Public SSH key。

```
cat ~/.ssh/id_rsa.pub
```

Windows 

```
用文字編輯器打開 “%homedrive%%homepath%.ssh\id_rsa.pub” 的內容
```




- #### Github 開專案

    - Public SSH key 複製下來貼到 Github 帳號 -> Account Settings -> SSH Keys 裡
    - 可以 push 和 pull 下來
